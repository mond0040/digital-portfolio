Mobile Application Design

DroneEntry is a drone startup company that provides a web application for drone pilots to manage portfolio, and for enterprises to manage projects effectively. We designed iOS mobile application interfaces for enterprise modules and for pilot views that are revealed on the enterprise modules. A primary target audience of this mobile application will be a new enterprise companies and it will be mainly used for managing pilots, projects, documents and clients on site.
